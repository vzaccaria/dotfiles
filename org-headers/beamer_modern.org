
# This is the master beamer file, updated with the latest goodies for beamer and org.

# We assume LUALATEX as the standard latex engine. It uses metropolis theme as base.

# Prerequisites
# 
# - Metropolis beamer theme; available in texlive 2016
#   
#       tlmgr init-usertree
#       tlmgr install beamertheme-metropolis
#
# - Emoji.sty package
# 
#   1. Download from here:  https://github.com/henningpohl/latex-emoji/blob/master/example.tex
#
#   2. Unpack in directory indicated by this command:
#   
#         kpsewhich -var-value=TEXMFHOME
#   
#   3. verify with kpsewhich.sty
# 
# - Fira Mono with ligatures (Linux install)
# 
#      mkdir -p ~/.local/share/fonts
#      for type in Bold Light Medium Regular Retina; do wget -O ~/.local/share/fonts/FiraCode-$type.ttf "https://github.com/tonsky/FiraCode/blob/master/distr/ttf/FiraCode-$type.ttf?raw=true"; done
#
#      fc-cache -f
#
# - Menlo (Unicode support)
#
#   Download and double click: 
# 
#     wget -O ~/.local/share/fonts/Menlo-Regular.ttf "https://github.com/hbin/top-programming-fonts/blob/master/Menlo-Regular.ttf?raw=true"
#     fc-cache -f
#
# - Pygmentize 
#
#   1. Install pip: sudo apt-get install python pip 
#   2. Install pygmentize: pip install Pygments
#      Remember: packages installed with pip are located in something like =/home/admin/.local/lib/python2.7/site-packages;= so this should be added to the path
